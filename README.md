# 🚀 DamirMedia Video Downloader Bot (v1.1.0)

Многофункциональный Telegram-бот для скачивания видео и аудио из популярных социальных сетей и видеохостингов.

Бот поддерживает загрузку файлов большого размера при использовании локального сервера Telegram Bot API!

## 🌟 Возможности

- **Поддержка множества сервисов:**
  - ▶️ YouTube
  - 🎵 TikTok
  - 📸 Instagram
  - 📺 VK / VK Clips
  - 📡 Rutube
  - 🟠 OK.ru
  - 🐦 Twitter / X
  - 📌 Pinterest
- **Выбор качества видео:** скачивание в форматах 360p, 720p, 1080p, либо в максимальном качестве.
- **Извлечение аудио:** скачивание видео в формате MP3.
- **Интерактивный UI:** отображение информации о видео (название, длительность, просмотры, превью) и удобные inline-кнопки.
- **Отображение прогресса:** обновление процентов загрузки прямо в сообщении.
- **Автоочистка:** автоматическое удаление временных скачанных файлов для экономии места на сервере (спустя 30 минут).
- **Защита от спама:** ограничение частоты запросов и запрет на параллельную загрузку от одного пользователя.

## 🛠 Технологии

- [Node.js](https://nodejs.org/)
- [Telegraf (v4)](https://telegraf.js.org/) — фреймворк для взаимодействия с Telegram API
- [youtube-dl-exec](https://github.com/microlinkhq/youtube-dl-exec) — обёртка над популярным загрузчиком, обеспечивающая скачивание медиа.

## ⚙️ Установка и запуск

### 1. Требования

- **Node.js** (рекомендуется v16+)
- **Local Telegram Bot API** (опционально, но **строго рекомендуется** для скачивания файлов свыше 50 МБ)

### 2. Клонирование репозитория

```bash
git clone https://github.com/zver1k/media-telegram-bot.git
cd media-telegram-bot
```

### 3. Установка зависимостей

```bash
npm install
```

### 4. Настройка переменных окружения

Создайте файл `.env` в корневой директории проекта:

```env
BOT_TOKEN=ваш_токен_бота_от_botfather
TG_API_BASE_URL=http://127.0.0.1:8081  # Адрес локального Telegram API сервера (если есть)
```
*Примечание: Если вы используете официальный сервер API, просто не указывайте `TG_API_BASE_URL`. В этом случае лимит на размер получаемых файлов для пользователей составит стандартные 50 МБ.*

### 5. Запуск

```bash
npm start
```
В консоли появится сообщение `🚀 DamirMedia Bot started` — бот готов принимать ссылки!

## 🐳 Развертывание Local Telegram Bot API (через Docker)

Для отправки пользователям файлов размером более 50 МБ необходимо использовать локальный сервер Telegram API. Самый простой способ его запустить — использовать Docker.

### 1. Получение API ID и API HASH
Вам понадобятся `TELEGRAM_API_ID` и `TELEGRAM_API_HASH`. 
Получите их, авторизовавшись на сайте [my.telegram.org](https://my.telegram.org/) (раздел **API development tools**).

### 2. Установка Docker
Если Docker еще не установлен на вашем сервере, выполните:
```bash
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
```

### 3. Запуск контейнера сервера
Создайте рабочую директорию и запустите официальный образ (в команде замените значения на свои ID и HASH):

```bash
mkdir -p /opt/telegram-bot-api-data

docker run -d -p 8081:8081 \
  --name=telegram-bot-api \
  --restart=always \
  -v /opt/telegram-bot-api-data:/var/lib/telegram-bot-api \
  -e TELEGRAM_API_ID="ваш_api_id" \
  -e TELEGRAM_API_HASH="ваш_api_hash" \
  aiogram/telegram-bot-api:latest -R /var/lib/telegram-bot-api --local
```

### 4. Настройка бота на работу с Local API
После успешного запуска сервера, убедитесь, что в файле `.env` вашего проекта указан правильный локальный адрес:

```env
TG_API_BASE_URL=http://127.0.0.1:8081
```

*Если бот до этого уже работал через стандартный `api.telegram.org`, сначала выполните запрос выхода (Log Out), открыв в браузере ссылку:*
`https://api.telegram.org/bot<ВАШ_ТОКЕН>/logOut`
